# Vimeo Speed

Chrome extension that automatically speeds up Vimeo videos.

## Features

- Set a default playback speed for all Vimeo videos
- Speed is automatically applied when videos start playing
- On-screen speed overlay shows the current speed for 2 seconds
- Works with embedded Vimeo players too
- Respects manual speed changes made through Vimeo's native controls

## Installation

### From Chrome Web Store
*(Coming soon)*

### Manual install
1. Download or clone this repository
2. Open `chrome://extensions/` in Chrome
3. Enable "Developer mode" (top right)
4. Click "Load unpacked" and select the project folder

## Usage

1. Click the extension icon to open the popup
2. Set your preferred default speed (0.5x - 4x)
3. Toggle the speed overlay on/off
4. All Vimeo videos will automatically play at your chosen speed
